# OpenTK.Platform

Platform support for OpenTK, contains backends for Windows, Linux, and macOS.

Also known as Platform Abstraction Layer 2 (PAL2).

Read more about how to use OpenTK at: https://opentk.net/

PAL2 specific documentation is currently hosted at: https://nogginbops.github.io/opentk.net/learn/pal2/Introduction.html
